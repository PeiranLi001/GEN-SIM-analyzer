How To Use
1 cmsrel CMSSW_10_2_22
2 cd CMSSW_10_2_22/src
3 cmsenv
4 git clone git@github.com:ram1123/GEN-SIM-analyzer.git
5 scramv1 b -j 8
6 cd GEN-SIM-analyzer
7 git submodule update --init
8 cd GenAnalyzer
9 # Befor running, check the config file and comment/uncomment line for MINIAOD/LHE-GEN
10 cmsRun python/ConfFile_cfg.py

Submit job with condor job

  1. To set proxy and create tarfile for your CMSSW directory:
1 python create_cmssw_tarfile.py
  1. Then modify output root file name (or output directory) in script RunGENSIM_condor.sh and output log file name in script RunGENSIM_condor.jdl
  2. Then submit condor job using command:
1 condor_submit RunGENSIM_condor.jdl
