This function return the sorted vector having cleaned AK4 jets. These jets are cleaned from photons as well as the AK8 jet collections.

Loop over all AK4 jets and selects a pair of jets having minimum delta mass wrt w-boson mass.

Select another pair of AK4 jets that when combined with the selected on-shell AK4 jets gives min delta M wrt the Higgs boson.

Vec_genJetAK8 is the vector of TLorentzVector that contains cleaned jets which is cleaned from photons as well as AK4 jets.

Select on-shell W-boson

Select off-shell W-boson from cleaned AK8 jets LV by skipping the on-shell w-boson.

3-jet category:

  • One AK8 jet and two AK4 jets.
  • Select one highest pT AK8 jets and two highest pT AK4 jets.

Selection of two AK4 jet and a AK8 jet.
